HFI Music Player

Reproductor de audio profesional para la web

Descripcion

HFI Music Player es un reproductor de audio web que permite cargar musica desde carpetas LOCALES, crear playlists personalizadas, importar/exportar listas M3U, y disfrutar de una experiencia de alta calidad con soporte para multiples formatos.

Formatos Soportados

MP3 FLAC WAV M4A OGG

Soporte especial para FLAC problematicos mediante fallback a Web Audio API.

Compatibilidad

Linux 100% compatible Chrome, Edge
Windows 10 100% compatible Chrome, Edge
Windows 11 100% compatible Chrome, Edge

Requiere navegador con soporte para File System Access API

Funciones Principales

  • Carga de carpetas locales con escaneo recursivo
  • Reproduccion con barra de progreso interactiva
  • Modo aleatorio (Shuffle)
  • Modo repeticion (Repeat)
  • Guardado automatico de sesion
  • Importacion/Exportacion de playlists M3U
  • Busqueda por titulo o album
  • Persistencia de volumen
  • Modo claro / Modo oscuro

Tecnologias

  • HTML5 / CSS3 / JavaScript Vanilla
  • Web Audio API (fallback FLAC)
  • File System Access API
  • MediaSession API
  • LocalStorage

Guia de uso

1
Haz clic en "Cargar musica" y selecciona una carpeta con archivos de audio
2
Las canciones apareceran en la biblioteca. Haz clic en cualquier cancion para reproducirla
3
Crea playlists desde el panel izquierdo y anade canciones usando el boton "+"
4
Usa SHF para reproduccion aleatoria y RPT para repetir la playlist
5
Importa archivos .m3u para crear playlists automaticamente
6
La sesion se guarda automaticamente cada 15 segundos
7
Usa el switch en la esquina superior derecha para cambiar entre modo claro y oscuro
